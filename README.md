# Suset.estylos E.I.R.L - Sistema de Gestión Cloud

Sistema desarrollado para el control de servicios y comisiones de la empresa **Suset.estylos E.I.R.L.** ubicada en Miraflores.

## 🚀 Características
- **Control de Acceso:** Autenticación segura vía Firebase.
- **Base de Datos en Tiempo Real:** Uso de Google Cloud Firestore.
- **Arquitectura Serverless:** Implementación gratuita y escalable.

## 🛠️ Tecnologías
- HTML5 / JavaScript ES6
- Tailwind CSS
- Firebase (Auth & Firestore)
